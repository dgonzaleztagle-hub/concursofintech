# ORIUNDO BEEPER FINANCIERO: El Centinela de tu Bolsillo

## 🎯 Propuesta de Valor
Oriundo no es solo una aplicación de auditoría; es un sistema de protección financiera proactivo diseñado para detectar abusos, educar al usuario y proporcionar herramientas legales inmediatas para recuperar su dinero.

## 🚀 Características del MVP (v2.5)
1. **Auditoría IA:** Análisis profundo de productos financieros usando Claude 3.5.
2. **Wallet Sentinel:** Integración simulada con billeteras virtuales (Google/Apple).
3. **Generador Legal:** Creación automática de cartas de renuncia basadas en la Ley 19.496.
4. **Accesibilidad Total:** Interfaz moderna y suave con Text-to-Speech para adultos mayores.
5. **Educación Proactiva:** Consejos contextuales de ciberseguridad y finanzas.

## 💼 Visión de Negocio
* **Target:** Usuarios bancarizados, adultos mayores y fintechs.
* **Modelo:** B2B (Integración con billeteras) o B2C (Suscripción por ahorro detectado).
* **Impacto:** Empoderamiento del ciudadano frente a instituciones financieras.
